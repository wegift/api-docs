---
title: "Get account balance"
openapi: "/reference/2024-02-05/openapi.json GET /balance"
description: "Fetch the current balances for your account"
---

Retrieves the account balance. Returns the balance in a specified currency if the currency parameter is provided. If not, returns all balances greater than 0.
